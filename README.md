# 🏙️ Ontro FPS

A high-performance, multiplayer browser-based First Person Shooter built with **Three.js** and **Socket.io**.

![Gameplay Screenshot](https://via.placeholder.com/800x450.png?text=Ontro+FPS+Gameplay)

## 🎮 Features
- **Tactical Combat**: 5 distinct weapons (Pistol, Assault Rifle, Shotgun, SMG, Sniper).
- **Pro Mechanics**: Dedicated ADS (Aim Down Sights), Sprinting, and individual weapon handling.
- **Urban Warfare**: Explore a handcrafted city map with skyscrapers, alleys, and industrial zones.
- **Multiplayer**: Real-time synchronization using WebSockets.
- **Customizable**: Built-in settings for FOV, Sensitivity, and Crosshair colors.

## 🚀 Quick Start
1. Clone the repo: `git clone https://github.com/d1tokirb/ontrofps.git`
2. Install dependencies: `npm install`
3. Start the server: `npm start`
4. Open `http://localhost:8000` in your browser.

## 🛠️ Tech Stack
- **Frontend**: Three.js (WebGL), Vanilla JS, CSS3.
- **Backend**: Node.js, Express, Socket.io.
- **Deployment**: Optimized for Vercel (Frontend) & Railway (Backend). Live at: [ontrofps-production.up.railway.app](https://ontrofps-production.up.railway.app)

## ⌨️ Controls
- **WASD**: Move
- **Shift**: Sprint
- **Space**: Jump
- **Left Click**: Shoot
- **Right Click**: ADS / Scope
- **1-5**: Switch Weapons
- **ESC**: Pause / Settings
